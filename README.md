# Clinic

   *Requests*
```
/ - main page

/patient/{id} - get patient by id

/patient/{id}/delete - delete patient

/patient/create - create patient

/patient/{id}/update - update patient

/patient/comments - get all comments

/patient/comment/{id} - get comment by id

/patient/comment/{id}/delete - delete comment

/patient/{id}/comment/create - create comment

/patient/comment/{id}/update - update comment
```
